# go_game_logger
I've been playing go for 20 years, stuck in rank limbo. I guess I can do something with that game to get out of coding tutorial hell.
That being said, this is a simple logger that can log moves in a game of Go. It's bare-bones for now, but I plan on continuing to work on it to eventually have it convert the log into an SGF (smart go format) file. But that's a ways away, and probably in a different language for portability's sake.
